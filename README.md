# 🌐 스마트 위치 전송 장갑 웹 서비스 (Smart Glove Web)

스마트 위치 전송 장갑 시스템의 **보호자용 웹 서비스**로, Node.js·Express와 EJS 템플릿을 사용해 위치 확인, 게시판, 보호자–피보호자 정보를 한 화면에서 관리할 수 있도록 구현했습니다.

---

## 🧩 웹 서비스 개요

- 피보호자의 위치·상태를 보호자가 웹에서 확인하고 관리할 수 있는 서비스  
- 회원가입·로그인, 보호자/피보호자 정보 관리, 위치 조회, 게시판 기능 제공  
- EJS 템플릿으로 공통 레이아웃과 각 기능 페이지를 분리해 유지보수성을 높였습니다.

---

## 🖥️ 화면 구성 (views)

- `midas.ejs`  
  - 서비스 메인 페이지로, 보호자와 피보호자 정보를 한눈에 보여주고 주요 메뉴(위치, 게시판, 마이페이지 등)로 이동할 수 있는 허브 역할을 합니다.

- `login.ejs`, `join.ejs`  
  - 보호자 계정 생성을 위한 회원가입 페이지와 로그인 페이지로, 입력 폼과 기본 유효성 검사를 포함해 인증 흐름의 시작점이 됩니다.

- `mypage.ejs`  
  - 로그인한 보호자의 프로필, 연락처, 피보호자 정보 등을 관리하는 마이페이지로, 추후 피보호자 장갑/앱 연동 정보 확장을 염두에 두고 설계했습니다.

- `board.ejs`, `post.ejs`, `form.ejs`  
  - 공지·소통용 게시판 목록, 게시글 상세, 글쓰기 화면으로 구성되며, 리스트–상세–작성 흐름이 자연스럽게 이어지도록 라우팅과 템플릿 구조를 맞추었습니다.

- `location.ejs`  
  - 피보호자의 위치를 조회하는 페이지로, 지도/리스트 UI와 연동될 것을 고려해 섹션 구조와 요소 배치를 설계했습니다.

- `about.ejs`, `nav.ejs`  
  - 서비스 목적과 사용 방법을 안내하는 소개 페이지와, 상단 공통 네비게이션으로 전체 페이지의 일관된 사용자 경험을 제공합니다.

---

## 🛠 사용 기술 (Web)

- **Backend**: Node.js, Express  
- **View**: EJS 템플릿 엔진, HTML5, CSS3  
- **기타**: Git, GitHub를 활용한 버전 관리 및 협업

---

## 👩‍💻 담당 역할

- 보호자용 웹 서비스 전체 화면 구조 설계(`midas.ejs` 중심 허브 + 기능별 뷰 분리)  
- 로그인/회원가입/마이페이지, 게시판, 위치 조회 등 주요 페이지의 마크업 및 템플릿 구성  
- 공통 네비게이션과 소개 페이지를 통해 서비스 흐름과 정보 구조를 설계하고, 확장 가능한 레이아웃 패턴을 적용했습니다.
